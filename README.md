# Kunal Jat Portfolio

Premium multi-page Next.js portfolio website for Kunal Jat, Senior UI/UX and Product Designer.

## Run

```bash
npm install
npm run dev
```

Then open:

```text
http://localhost:3000
```

## Structure

- `app/` - Next.js App Router pages and dynamic case-study routes
- `components/` - reusable UI, motion, hero, navigation, and section components
- `data/projects.js` - portfolio content, project metrics, and case-study data
- `app/globals.css` - Tailwind layers, glassmorphism, animated background, and design tokens
- `public/Kunal_Jat_Resume.pdf` - downloadable resume

## Pages

- Homepage sections: hero, about, skills, projects, gallery, experience, contact, footer
- Case studies:
  - Restaurant Management System
  - RIMS
  - PDF Seva
  - SPSU Website Redesign
